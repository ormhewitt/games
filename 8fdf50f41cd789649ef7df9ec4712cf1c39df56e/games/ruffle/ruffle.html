What is Ruffle?

ruffle is a Flash Player emulator built in the Rust programming language. It allows Flash games to run after the deprecation of Flash player.

Is it free?

It is completely free - many of the games in the archive run on ruffle through my server.
However, for some games, it is not practical to do so. For these games, you need to install the Ruffle plugin in your browser.

How do I download it?

To download it, visit the ruffle website by clicking this link. Click the name of your browser to download the correct version.
Alternatively, select the name of your browser below, and then follow the instructions to install the extension.

Firefox . Chrome / Edge / Safari

Installation instructions

Chrome / Edge / Safari

  1. First, go to Chrome settings using the 3 dots on the top right corner.
  2. Then select 'extensions'
  3. Flip the developer mode toggle so that it is turned on
  4. Press load unpacked and select the zip folder from the file explorer that pops up.
  5. Your extension will now be installed - make sure that the toggle next to it is turned on.

Firefox

  1. Open Firefox, and go to Add-ons on the top menu.
  2. Drag and drop the xpi file onto the list of currently available extensions.
  3. Click 'install', and then restart the browser when prompted to activate the extension.
